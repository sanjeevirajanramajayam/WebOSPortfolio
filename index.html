My Projects

When I was kid, I always really liked computers.
My parents invested early in computers and I have been using them since I was five.

When I was in the sixth grade, I got really into retro gaming. In particular, emulating retro games.
Through this I was introuduced to the Raspberry Pi 3B.
I asked my uncle to get me one and was introuduced to Linux.

Then during lockdown, I was stuck at home.
I learnt Python through freeCodeCamp videos.
I distinctly remember writing my first print and if statement and showing my parents.

When I was in the tenth grade, I bugged my parents to get me an old Dell PC so that I could install linux on it safely.
It soon became my daily driver running void linux.

Then after my 12th board exams, I created a full stack project with django and a locally hosted online pong game using pygame.

Then at college, I was introuduced to full-stack development and leetcode.
My focus shifted to placements which is where it stands now.
